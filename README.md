## Dependencies
The app is built in ionic so it needs everything ionic needs:
- [node and npm](https://nodejs.org/en/)
- cordova
- ionic
    npm install -g cordova ionic

## Run
To run the project in browser:
    ionic serve

To run the project on ios emulator:
    ionic build ios
    ionic serve ios
